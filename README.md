# Quasar EOS Wallet
This is a specialized EOS wallet that you can use to start buildling a dapp for a profit sharing company.

# Features:
* Authorize users with Scatter - login, logout, get user account name
* Get user balance in project token, EOS and EOSDT
* Get balance for presale contract (how many native tokens are left for presale) and profits accumulated in EOSDT
* Get token supply info to calculate user's share in the company
* Show users balance and share in the business
* Purchase share in the company by sending EOSDT tokens to presale contract
* Show history of all share purchases
* Show history of all token issues
* Show history of dividends received and total of dividends received from the company
* Show users EOSDT balance

# Quasar EOS Wallet
Специализированный EOS кошелек, как пример и быстрый старт для создания личного кабинета инвестора в компании, выплачивающей дивиденды
С помощью данного проекта вы сможете создать приложение, которое будет служить личным кабинетом для инвесторов и вашим средством взаимодействия с ними.

# Функционал:
* Авторизация через Scatter - login, logout, получение имени пользователя
* Форма покупки доли в комании, генерация транзакции, подписание через Scatter
* Получение баланса в токене проекта и EOSDT (USD) для пользователя, контракта предпродажи, контракта с прибылью
* Автоматическое получение обновлений баланса через Dfuse.io и их отображение
* Получение общей информации о токене - кол-во выпущенных токенов всего
* Получение истории выпуска токена и ее отображение в виде таблицы
* Получение истории покупок токена пользователем и ее отображение в таблице
* Получение истории получения дивидендов и отображение. Подсчет общей суммы полученной от контракта с дивидендами в EOSDT

Используемые библиотеки для EOS:
* ScatterJS
* Dfuse
